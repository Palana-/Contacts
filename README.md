# 电话本

电话本是一个原生 Android 通讯录应用，使用 Kotlin 和 Android View 实现，不依赖 Flutter 或跨平台框架。

## 功能

- 应用名称：电话本
- 应用图标：本子形状图标
- 首页顶部标题栏：新增联系人、更多功能菜单
- 通讯录列表：
  - 有头像联系人使用两列网格展示
  - 无头像联系人使用紧凑列表展示
  - 联系人按姓名首字母排序，中文按拼音首字母排序
- 联系人详情：
  - 点击联系人弹出详情弹窗
  - 显示头像、姓名、号码、拨号、编辑、删除
  - 删除联系人前二次确认
- 新增/编辑联系人：
  - 独立 Activity 页面
  - 手机号必填
  - 姓名和头像至少填写一个
- 手机通讯录同步：
  - 从手机通讯录导入
  - 写入手机通讯录
  - 头像以 APP 内数据为主
- 迁移包：
  - 导出 `.pbk` 迁移包
  - 导入 `.pbk` 迁移包
  - 迁移包包含联系人数据和原始头像文件，避免通过 SIM 卡迁移造成头像失真

## 构建

```powershell
cd android
.\gradlew.bat assembleDebug
```

Debug APK 输出位置：

```text
android/app/build/outputs/apk/debug/app-debug.apk
```
